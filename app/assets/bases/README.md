# Base icons

The specific-base picker (and the goal card) shows each item base's in-game icon
from here. Files mirror the `img` paths in `data/poe2_item_bases.json`, e.g.

```
app/assets/bases/Weapons/TwoHandWeapons/Staves/FourQuarterstaff1.webp
```

These are **not** generated by the data build — download them with:

```bash
cd pipeline
python fetch_base_images.py
```

(run from a machine that can reach craftofexile.com). The app references
`assets/bases/<img>` and falls back to an "image pending" placeholder for any
file that isn't present, so it works fine before/while the icons are fetched.

Icons are © Grinding Gear Games, surfaced via Craft of Exile.
